## Activation rule

Distribution remains disabled until every applicable blocker is closed with a
dated source, owner, implementation evidence, public copy, and approval record.
Regional law changes quickly; verify official sources on the activation date.

Do not use a residential address in a storefront, privacy notice, representative
listing, trader disclosure, or other public record. If a region requires a public
address or representative, obtain an appropriate non-residential solution before
activation.
