This directory is the working evidence package for AgentKip's App Store and
TestFlight submissions. It is not a substitute for the answers entered in App
Store Connect. Every answer must be reconciled against the exact release build,
its embedded SDKs, the production website, and the deployed Noggin configuration.

## Submission rule

Do not turn a candidate answer into an App Store Connect declaration until its
evidence field contains a release commit, build number, configuration, and test
result. A blank or `BLOCKED` field is a release blocker, not permission to choose
the most favorable answer.
