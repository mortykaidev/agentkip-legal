# Control Layer

This directory records the facts, versions, evidence, placement, and release gates that keep
AgentKip's legal copy synchronized with the product. Public documents live in
`10-public-legal/`; approved exports are generated from the manifest and are not hand-edited.

## Initial claim

The baseline implementation covers the complete launch package, in-app copy, Apple submission
worksheets, internal compliance operations, inactive regional templates, user guides, and
future-review triggers described in the draft pull request.
